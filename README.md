# 🧮 ReactJS Calculator

Una calculadora interactiva y moderna construida con **React**. Este proyecto demuestra el manejo de estados, eventos de usuario y lógica aritmética básica en un entorno web dinámico.

## 🚀 Características

* **Operaciones Básicas:** Suma, resta, multiplicación y división.
* **Interfaz Responsiva:** Diseño adaptable a diferentes tamaños de pantalla.
* **Manejo de Estado:** Uso de `useState` para gestionar los operandos y resultados.
* **Limpieza de Pantalla:** Función *Clear* para reiniciar los cálculos.

## 🛠️ Tecnologías Utilizadas

* **ReactJS** (Hooks)
* **CSS3** (Flexbox / Grid)
* **JavaScript** (ES6+)

## 📦 Instalación y Uso

Sigue estos pasos para ejecutar el proyecto localmente:

1.  **Clona el repositorio:**
    ```bash
    git clone https://github.com/tu-usuario/nombre-del-repo.git
    ```

2.  **Entra al directorio:**
    ```bash
    cd nombre-del-repo
    ```

3.  **Instala las dependencias:**
    ```bash
    npm install
    ```

4.  **Inicia la aplicación:**
    ```bash
    npm start
    ```
    > La aplicación se abrirá automáticamente en `http://localhost:3000`.

## 📂 Estructura de la Carpeta `src`

```text
src/
├── components/       # Componentes reutilizables (Botones, Pantalla)
├── hooks/            # Lógica personalizada (opcional)
├── App.js            # Componente principal
├── App.css           # Estilos generales
└── index.js          # Punto de entrada de React
```
